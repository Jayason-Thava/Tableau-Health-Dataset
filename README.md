- **Chart Type:** Line Chart

### 3. Population Distribution by Gender (Latest Year for a Selected Country)
- **Chart Type:** Pie Chart  
- **Details:** Population distribution by gender for a selected country.  
- **Features:**  
  - Country filter (dropdown, single value).  
  - Filtered to most recent year.  
  - Population shown as percentage of total.  
- **Deliverable:** Worksheet named *"Population by Gender"*.

---

### 4. Health Metric Comparison (Scatter Plot)
- **Chart Type:** Scatter Plot  
- **Details:** Relationship between Average Life Expectancy and Average BMI.  
- **Features:**  
  - Life Expectancy vs BMI axes.  
  - Country on Detail (points).  
  - Continent on Color (differentiation).  
  - Continent filter available.  
- **Deliverable:** Worksheet named *"Life Expectancy vs BMI"*.
